ADAT
===============

###### 功能：
1. 抓包。
2. 记录收到的短信。

###### 备注
* 抓包内容和短信日志在 SDCARD/ADAT目录下面。
* 抓包使用 tcpdump，可以用 wireshark 解析。

—————————————————————————————————————————

###### Function
1. dump network.
2. log sms that receive.

###### Annotations
* Dump file and sms log file is at sdcard/adat.
* Dump file is pcap file, it can be parsed with wireshark.
